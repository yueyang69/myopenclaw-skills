# Deep Learning Guide - 深度学习指导技能

**版本：** 2.0 (双模型协作版)  
**创建日期：** 2026-03-14  
**架构：** 参考 auto-task-runner v3

---

## 📖 技能描述

帮助用户进行**系统性、深入的学习**，避免"入宝山空手归"。

**核心特性：**
- 🧠 双模型协作（Qwen 规划 + Claude 审核）
- 📍 知识地图自动生成
- 📅 30 天学习计划
- 🔄 复习题生成
- 📝 学习日志模板

---

## 🎯 使用方式

### 命令行
```bash
cd /home/admin/.openclaw/workspace/skills/deep-learning-guide

# 启动深度学习
python3 main.py deep_learn [主题]

# 查看进度
python3 main.py learn_status [主题]

# 生成复习题
python3 main.py learn_review [主题]
```

### 自然语言触发
- "我想深入学习 XXX"
- "帮我制定 XXX 的学习计划"
- "XXX 怎么学才能精通"

## 📋 学习框架

### 阶段 1：知识地图（AI 生成）
- 核心概念（5-8 个）
- 学习路径（入门→进阶→精通）
- 推荐资源（每个阶段 2-3 个）
- 里程碑（3 个可验证输出）

### 阶段 2：刻意练习（用户执行）
- 费曼技巧
- 项目驱动
- 主动回忆
- 间隔重复

### 阶段 3：体系整合（AI 辅助）
- 知识图谱
- 个人笔记
- 公开输出

### 阶段 4：实战验证（用户执行）
- 开源项目
- 工作应用
- 教学分享

## 🚫 避免的陷阱

| 陷阱 | 解决方案 |
|------|---------|
| 收藏家综合征 | 只选 1 个主教材 |
| 教程地狱 | 每学 10 分钟，动手 30 分钟 |
| 完美主义 | 70% 理解就前进 |
| 碎片学习 | 按知识地图顺序 |
| 只输入不输出 | 强制输出任务 |
